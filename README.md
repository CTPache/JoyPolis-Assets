# Ace Attorney Joypolis Assets Recreation

A comprehensive collection of recreated assets from the Ace Attorney interactive attractions at SEGA's Joypolis amusement centers in Japan.

## About the Project

This repository preserves and recreates visual assets from two unique Ace Attorney attractions that featured original stories and exclusive content:

- **逆転裁判 in Joypolis** (Gyakuten Saiban in Joypolis) - Phoenix Wright attraction
- **逆転検事 in Joypolis** (Gyakuten Kenji in Joypolis) - Miles Edgeworth attraction

These interactive experiences allowed fans to participate in original cases set in the Ace Attorney universe, featuring new characters, storylines, and courtroom drama.

## The Attractions

### Gyakuten Saiban in Joypolis (Phoenix Wright)

Featured three original cases with exclusive characters and storylines:

1. **Episode 1: Turnabout Fly High** (飛翔)
   - New characters: Nakaki Kurumi (victim), featuring Maggey Byrde and Frank Shawit

2. **Episode 2: Turnabout Locked Room** (密室)
   - New characters: Rando Daiba (victim), featuring Wendy Oldbag, Ini Miney, and Will Powers

3. **Episode 3: Turnabout Don** (逆転のドン)
   - New characters: Yumi Temgashiwa (victim) and Kogyoin Don (antagonist)
   - Also featuring Desirée DeLite and Sal Manella

### Gyakuten Kenji in Joypolis (Miles Edgeworth)

Featured three original episodes:

1. **EP-1: Turnabout Symbol** (逆転の紋章)
2. **EP-2: Turnabout Message** (逆転のメッセージ)
3. **EP-3: Turnabout Theme Park** (逆転のテーマパーク)

## Asset Creation

Assets in this repository were recreated using:
- **Nano Banana Pro** - Primary generation tool
- **GIMP** - Image editing and refinement

Source materials include:
- Promotional materials from Ameba.jp and social media
- On-site photographs and documentation
- Attraction footage and gameplay videos
- Official Joypolis booklets and promotional items

## File Organization

- **PNG/JPG files**: Final exported assets
- **README.md files**: Documentation in each folder explaining the contents
